# CGPA Calculator

A semester-based CGPA calculator built specifically 
for Nigerian university students.

## Features
- Enter multiple courses with grades and units
- Automatically converts marks to grade points
- Calculates weighted CGPA per semester
- Saves results to a personal file by semester

## How to run
1. Compile: g++ cgpa.cpp -o cgpa
2. Run: ./cgpa

## What I learned
Weighted calculations, const references, vector 
parallelism, setprecision formatting and fstream.
